1- O que é um repositório? 
   Repositório é um espaço que é usado para arquivar coisas diferentes.

2- Qual a diferença entre Git e GitHub?
   Git é um software onde vários desenvolvedores podem trabalhar ao mesmo tempo. GitHub é uma rede social de desenvolvedores.

3- Porque utilizar controle de versão para os arquivos de um projeto de desenvolvimento de software?
   Pois podemos organizar o processo, ver quando uma pessoa faz uma modificação e podendo atualiza-lo.
